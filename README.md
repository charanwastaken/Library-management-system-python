# Library Management System
Academic project developed during 12th grade as part of computer science coursework.

Built a GUI-based Library Management System using Python, Tkinter, file handling, and MySQL. Designed interactive screens with buttons for login, viewing available books, borrowing and returning books, and updating library records. The project focused on strengthening Python fundamentals, GUI event handling, and database connectivity.

## Features
- Add, update, delete, and display books
- Issue and return books
- Store user and borrowing data
- Persistent storage using CSV files and MySQL database

## Technologies Used
- Python
- MySQL
- File Handling (CSV)
- MySQL Connector

## How to Run
1. Install Python
2. Install MySQL and create a database
3. Run `setup_database.py`
4. Run `main.py`

## Purpose
This project was developed as an academic project to strengthen Python programming, database connectivity, and logical problem-solving skills.
